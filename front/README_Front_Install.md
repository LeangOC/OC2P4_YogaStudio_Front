## Installation,lancement et tests

### 1. Clone
> git clone https://github.com/LeangOC/OC2P4_YogaStudio_Front.git

### 2. Dépendances Angular :  
> $ npm install  

### 3. Démarrage de l'application :
> $ ng serve

### 4. Tests unitaires Jest :
> $ npm run test

### 5. Tests e2e cypress avec coverage:  
> $ ng run yoga:serve-coverage

> $ npx cypress run  

> $ npm run e2e:coverage
